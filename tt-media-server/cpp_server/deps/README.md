# C++ dependencies

- **tt-metal** — Required submodule. After cloning the repo, run from repo root:
  ```bash
  git submodule update --init --recursive
  ```
  Then build and install tt-metal into `tt-metal/install`; see cpp_server README “Optional: tt-metal”.

Build and install outputs (`tt-metal/build/`, `tt-metal/install/`) are gitignored.
