## Patent Grant Clarification

The Apache 2.0 license includes an explicit patent grant (Section 3). This means:

1. **Contributors grant you patent rights:** If a contributor's code uses their
   patent, they automatically grant you a license to use that patent (for this
   software only).

2. **Defensive termination:** If you sue anyone claiming this software infringes
   your patents, your patent license terminates automatically.

3. **Why this matters:** This protects users and contributors from patent
   litigation while encouraging open collaboration.
